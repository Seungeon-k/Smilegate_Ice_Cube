# RecalculateClipping

## 객체

> [TextMeshProUGUI](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/TextMeshProUGUI)

## 설명

이 함수는 텍스트 메쉬 프로 UI 요소의 클리핑을 재계산합니다. 클리핑이 변경된 경우 이 함수를 호출하여 UI 요소가 올바르게 표시되도록 할 수 있습니다. 이 함수는 인수가 없으며, 호출 후에는 UI 요소의 시각적 상태가 업데이트됩니다.

## 함수

RecalculateClipping()

없음

### 반환값

없음

## 예제 코드

```lua
TextMeshProUGUI:RecalculateClipping()
```
