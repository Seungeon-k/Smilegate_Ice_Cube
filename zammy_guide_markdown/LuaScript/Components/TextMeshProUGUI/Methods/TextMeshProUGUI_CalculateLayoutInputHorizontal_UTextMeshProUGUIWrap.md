# CalculateLayoutInputHorizontal

## 객체

> [TextMeshProUGUI](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/TextMeshProUGUI)

## 설명

이 함수는 수평 레이아웃 입력을 계산합니다. 주로 UI 요소의 레이아웃을 조정할 때 사용됩니다. 이 메서드는 반환값이 없으며, 호출 시 특별한 인수는 필요하지 않습니다. 사용 시 주의할 점은 UI 요소의 상태에 따라 레이아웃 계산 결과가 달라질 수 있다는 것입니다.
