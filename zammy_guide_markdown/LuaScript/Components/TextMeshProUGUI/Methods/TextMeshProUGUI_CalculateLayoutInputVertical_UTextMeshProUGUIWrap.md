# CalculateLayoutInputVertical

## 객체

> [TextMeshProUGUI](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/TextMeshProUGUI)

## 설명

이 함수는 텍스트 메쉬 프로 UI 요소의 수직 레이아웃 입력을 계산합니다. 이 메서드는 레이아웃 시스템에서 호출되며, UI 요소의 크기와 위치를 결정하는 데 사용됩니다. 이 함수는 반환값이 없으며, 주로 레이아웃 업데이트 과정에서 자동으로 호출됩니다.

## 함수

CalculateLayoutInputVertical()
