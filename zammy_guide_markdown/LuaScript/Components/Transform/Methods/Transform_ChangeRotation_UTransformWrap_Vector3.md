# ChangeRotation

## 객체

> [Transform](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Transform)

## 설명

rotation을 변경하고 동기화를 진행합니다.

## 함수

ChangeRotation(angles)

### 매개변수

| **형식** | **파라미터** | **설명** |
| --- | --- | --- |
| [`Vector3`](https://developers-zammysmith.onstove.com/ko/LuaScript/DataTypes/Vector3) | `angles` | 변경할 회전각도 |

### 반환값

없음

## 예제 코드

```lua
transform.ChangeRotation(newAngles)
```
