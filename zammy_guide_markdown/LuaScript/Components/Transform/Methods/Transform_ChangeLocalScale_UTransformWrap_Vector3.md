# ChangeLocalScale

## 객체

> [Transform](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Transform)

## 설명

local scale 값을 변경하고 동기화를 진행합니다.

## 함수

ChangeLocalScale(scale)

### 매개변수

| **형식** | **파라미터** | **설명** |
| --- | --- | --- |
| [`Vector3`](https://developers-zammysmith.onstove.com/ko/LuaScript/DataTypes/Vector3) | `scale` | 변경할 로컬 스케일 |

### 반환값

없음

## 예제 코드

```lua
transform.ChangeLocalScale(newScale)
```
