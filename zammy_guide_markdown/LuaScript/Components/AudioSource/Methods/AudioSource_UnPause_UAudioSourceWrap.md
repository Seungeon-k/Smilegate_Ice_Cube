# UnPause

## 객체

> [AudioSource](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/AudioSource)

## 설명

이 함수는 일시 정지된 오디오 소스의 재생을 다시 시작합니다. 오디오 소스가 일시 정지 상태일 때 호출하면, 재생이 중단된 지점부터 오디오가 재개됩니다. 이 함수는 인수가 필요하지 않으며, 오디오 소스가 이미 재생 중인 경우에는 아무런 효과가 없습니다.

## 함수

UnPause()
