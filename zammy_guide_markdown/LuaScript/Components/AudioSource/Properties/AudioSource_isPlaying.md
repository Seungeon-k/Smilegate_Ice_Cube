# isPlaying

## 객체

> [AudioSource](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/AudioSource)

## 설명

이 프로퍼티는 현재 오디오 소스가 클립을 재생 중인지 여부를 반환합니다. 읽기 전용이며, 값을 설정할 수 없습니다. 이 프로퍼티를 사용하여 오디오 소스의 재생 상태를 확인할 수 있습니다.

## 프로퍼티 정의

- **이름**: `isPlaying`
- **타입**: `boolean`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ❌

## 사용 예제

```lua
isPlaying = audioSource.isPlaying
```
