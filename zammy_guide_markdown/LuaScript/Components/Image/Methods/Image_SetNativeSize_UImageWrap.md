# SetNativeSize

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 함수는 이미지의 원래 크기로 크기를 조정합니다. 이 메서드를 호출하면 이미지의 크기가 해당 이미지의 원본 크기로 설정됩니다. 사용 시 주의할 점은, 이 메서드는 이미지가 로드된 후에 호출해야 하며, 그렇지 않으면 예상치 못한 결과가 발생할 수 있습니다.

## 함수

SetNativeSize()

없음

없음

```lua
Image:SetNativeSize()
```
