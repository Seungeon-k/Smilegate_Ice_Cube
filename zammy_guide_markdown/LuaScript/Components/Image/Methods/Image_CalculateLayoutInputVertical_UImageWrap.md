# CalculateLayoutInputVertical

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 함수는 UI 요소의 수직 레이아웃 입력을 계산하는 데 사용됩니다. 이 메서드는 레이아웃 시스템에서 UI 요소의 크기와 위치를 결정하는 데 필요한 정보를 업데이트합니다. 호출 후에는 UI 요소의 레이아웃이 적절하게 조정됩니다. 이 함수는 주로 레이아웃 업데이트가 필요한 경우에 사용됩니다.
