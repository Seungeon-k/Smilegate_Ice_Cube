# CalculateLayoutInputHorizontal

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 함수는 수평 레이아웃 입력을 계산하는 데 사용됩니다. 주로 UI 요소의 레이아웃을 조정할 때 호출됩니다. 이 함수는 반환값이 없으며, 호출 후 UI 요소의 레이아웃이 업데이트됩니다. 사용 시 주의할 점은 이 함수를 호출하기 전에 UI 요소가 적절히 초기화되어 있어야 한다는 것입니다.

## 함수

CalculateLayoutInputHorizontal()
