# DisableSpriteOptimizations

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 함수는 스프라이트 최적화를 비활성화합니다. 스프라이트 최적화는 성능을 향상시키기 위해 사용되며, 이 함수를 호출하면 해당 최적화가 해제됩니다. 이 함수는 인수가 필요하지 않으며, 호출 후에는 스프라이트의 최적화가 적용되지 않습니다.

## 함수

DisableSpriteOptimizations()

### 매개변수

없음

### 반환값

없음

## 예제 코드

```lua
Image:DisableSpriteOptimizations()
```
