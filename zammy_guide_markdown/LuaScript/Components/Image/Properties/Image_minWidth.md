# minWidth

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 프로퍼티는 이미지의 최소 너비를 반환합니다. 이 값은 읽기 전용이며, 설정할 수 없습니다.

## 프로퍼티 정의

- **이름**: `minWidth`
- **타입**: `number`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ❌

## 사용 예제

```lua
local minWidth = image.minWidth
```
