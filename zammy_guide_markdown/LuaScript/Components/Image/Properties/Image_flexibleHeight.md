# flexibleHeight

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 프로퍼티는 이미지의 유연한 높이를 나타냅니다. 이 값은 이미지의 레이아웃에서 자동으로 조정되는 높이를 결정하는 데 사용됩니다.

이 프로퍼티는 읽기 전용이며, 설정할 수 없습니다. 따라서 값을 변경하려면 다른 방법을 사용해야 합니다.

## 프로퍼티 정의

- **이름**: `flexibleHeight`
- **타입**: `number`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ❌

## 사용 예제

```lua
local height = image.flexibleHeight
```
