# fillOrigin

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

`fillOrigin` 프로퍼티는 이미지의 채우기 방향을 설정하는 데 사용됩니다. 이 프로퍼티는 정수형 값을 가지며, 이미지의 채우기 방향을 결정하는 데 필요한 값을 설정할 수 있습니다. 사용 시 주의할 점은, 이 프로퍼티의 값이 유효한 범위 내에 있어야 하며, 잘못된 값이 설정될 경우 예외가 발생할 수 있습니다.

## 프로퍼티 정의

- **이름**: `fillOrigin`
- **타입**: `number`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ✅

## 사용 예제

```lua
image.fillOrigin
```
