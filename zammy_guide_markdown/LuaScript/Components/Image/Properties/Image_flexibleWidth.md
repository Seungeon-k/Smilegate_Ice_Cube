# flexibleWidth

## 객체

> [Image](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/Image)

## 설명

이 프로퍼티는 이미지의 유연한 너비를 반환합니다. 이 값은 이미지의 레이아웃을 조정하는 데 사용됩니다. 프로퍼티는 읽기 전용이며, 설정할 수 없습니다. 사용 시 주의할 점은 이 값이 동적으로 변경될 수 있으므로, 필요에 따라 적절한 업데이트가 필요할 수 있습니다.

## 프로퍼티 정의

- **이름**: `flexibleWidth`
- **타입**: `number`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ❌

## 사용 예제

```lua
local width = image.flexibleWidth
```
