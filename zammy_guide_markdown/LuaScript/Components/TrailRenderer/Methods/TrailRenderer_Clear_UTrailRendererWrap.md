# Clear

## 객체

> [TrailRenderer](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/TrailRenderer)

## 설명

이 함수는 TrailRenderer의 모든 포인트를 제거합니다. 새로운 위치에서 트레일을 다시 시작할 때 유용합니다. 이 메서드는 포인트를 초기화하므로, 호출 후에는 새로운 트레일을 생성해야 합니다.

## 함수

Clear()
