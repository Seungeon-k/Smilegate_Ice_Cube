# startColor

## 객체

> [TrailRenderer](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/TrailRenderer)

## 설명

이 프로퍼티는 트레일의 시작 부분에서 사용할 색상을 설정합니다. 색상은 `Color` 타입으로 반환되며, 설정할 수 있습니다. 이 프로퍼티를 사용하여 트레일의 시각적 효과를 조정할 수 있습니다.

트레일의 시작 색상을 변경하면, 트레일이 생성될 때 해당 색상이 적용됩니다. 이 프로퍼티는 읽기와 쓰기가 모두 가능하므로, 필요에 따라 색상을 동적으로 변경할 수 있습니다.

## 프로퍼티 정의

- **이름**: `startColor`
- **타입**: `Color`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ✅

## 사용 예제

```lua
trailRenderer.startColor
```
