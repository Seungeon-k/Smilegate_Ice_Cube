# proceduralSimulationSupported

## 객체

> [ParticleSystem](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/ParticleSystem)

## 설명

이 프로퍼티는 해당 시스템이 절차적 시뮬레이션을 지원하는지를 나타냅니다. 이 값을 통해 시뮬레이션 방식에 대한 선택을 할 수 있습니다. 이 프로퍼티는 읽기 전용이며, 값을 설정할 수 없습니다.

## 프로퍼티 정의

- **이름**: `proceduralSimulationSupported`
- **타입**: `boolean`
- **읽기 가능(Read)**: ✅
- **쓰기 가능(Write)**: ❌

## 사용 예제

```lua
local isSupported = particleSystem.proceduralSimulationSupported
```
