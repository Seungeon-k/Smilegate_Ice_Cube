# ForceUpdateRectTransforms

## 객체

> [RectTransform](https://developers-zammysmith.onstove.com/ko/LuaScript/Components/RectTransform)

## 설명

이 함수는 RectTransform의 내부 데이터를 강제로 재계산합니다. 주로 UI 요소의 위치나 크기 변경 후에 호출하여, 레이아웃을 즉시 업데이트할 필요가 있을 때 사용됩니다. 이 함수는 인수가 없으며, 호출 후에는 RectTransform의 상태가 갱신됩니다.

## 함수

ForceUpdateRectTransforms()

### 매개변수

없음

### 반환값

없음

## 예제 코드

```lua
RectTransform:ForceUpdateRectTransforms()
```
