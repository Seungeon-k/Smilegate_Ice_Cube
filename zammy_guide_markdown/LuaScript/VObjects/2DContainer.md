# 2DContainer

## 모듈

> VFramework

## 개요

2DContainer는 2D 기반 오브젝트들을 정리하기 위해 제공되는 폴더형 컨테이너입니다.  

UIObject나 NamePlate처럼 화면 UI 계층에 포함되는 요소들을 한 공간에 모아두기 위한 구조적 엔티티이며,  

자체적인 기능이나 동작은 없습니다.  

렌더링·물리·이벤트 처리에 관여하지 않고, 단순히 하위 2D 요소들을 분류하고 워크스페이스에서 UI 전용 영역을 구성하는 역할만 수행합니다.
